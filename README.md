# Sistem Informasi Manajemen Perjanjian Kerjasama

- **Deskripsi sistem** — penjelasan tujuan SIM-PKS
- **Stack teknologi** — Backend (CodeIgniter 4), Frontend (React + TypeScript), JWT, Chart.js, dll.
- **14 fitur utama** yang mencakup seluruh service yang ditemukan:
  1. Autentikasi & Otorisasi (JWT)
  2. Manajemen Pengguna
  3. Manajemen Role & Hak Akses
  4. Manajemen Fakultas
  5. Manajemen Program Studi
  6. Manajemen Dekan
  7. Manajemen Kontak Institusi
  8. Manajemen Dosen
  9. Manajemen Organisasi Mitra
  10. Manajemen Perjanjian Kerjasama (termasuk filter status: Progress/Deadline/Complete/Hold)
  11. Statistik & Grafik Kerjasama
  12. Manajemen Implementasi
  13. Manajemen Dokumentasi
  14. Manajemen Tipe Dokumen
